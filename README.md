## Подсчёт слов в тексте

### Назначение программы
Программа анализирует текстовую строку, введённую пользователем, подсчитывает частоту каждого слова и сохраняет результат в файл.

### Алгоритм работы

1. **Ввод строки**  
   Пользователь вводит произвольную строку текста с клавиатуры.

2. **Нормализация**  
   Все символы приводятся к нижнему регистру. Это необходимо, чтобы слова, отличающиеся только регистром (например, «Cat» и «cat»), считались одинаковыми.

3. **Разбиение на слова**  
   Строка разделяется на отдельные слова по пробельным символам (пробелы, табуляции, переносы строк). Полученный список слов сохраняется.

4. **Подсчёт частоты**  
   Создаётся пустой словарь (ассоциативный массив). Каждое слово из списка поочерёдно обрабатывается:
   - Если слово уже присутствует в словаре, значение счётчика для этого слова увеличивается на 1.
   - Если слово встречается впервые, оно добавляется в словарь с начальным значением 1.

5. **Вывод на экран**  
   Для каждого уникального слова программа выводит строку в формате:  
   `слово -> количество`

6. **Сохранение в файл**  
   Аналогичные пары «слово → количество» записываются в текстовый файл `result.txt` (кодировка UTF-8). Если файл с таким именем уже существует, он будет перезаписан.

### Пример работы

**Входные данные:** 

Введите строку: cat dog cat bird dog cat


**Вывод на экран:** 

cat -> 3

dog -> 2

bird -> 1

Результат сохранён в result.txt


**Содержимое файла `result.txt`:** 

cat -> 3

dog -> 2

bird -> 1


### Тестирование программы

| Входная строка                     | Ожидаемый вывод на экран                          | Результат |
|------------------------------------|---------------------------------------------------|-----------|
| `a a a`                            | `a -> 3`                                          | ✔️ |
| `one two three`                    | `one -> 1`<br>`two -> 1`<br>`three -> 1`          | ✔️ |
| `Hello hello HELLO`                | `hello -> 3`                                      | ✔️ |
| `собака кот кот`                   | `собака -> 1`<br>`кот -> 2`                       | ✔️ |
| `word` (одно слово)                | `word -> 1`                                       | ✔️ |
| Пустая строка (просто Enter)       | Ничего не выводится, файл создаётся пустым        | ✔️ |

### Особенности реализации

- Регистронезависимость – слова `cat`, `Cat`, `CAT` считаются одинаковыми.
- Разделителем слов служит любой пробельный символ (в том числе несколько пробелов подряд).
- Знаки препинания (запятые, точки, восклицательные знаки и т.п.) не удаляются. Например, слова `"cat"` и `"cat,"` будут считаться разными. 
- Результат сохраняется в файл `result.txt` в той же директории, откуда запущена программа.

### Заключение

Разработанная программа демонстрирует практическое применение словаря (`map`) для классической задачи подсчёта частоты слов. Алгоритм работы состоит из последовательных этапов: ввод строки, приведение к нижнему регистру, разбиение на слова, итеративное заполнение словаря с накоплением счётчиков, вывод результатов на экран и сохранение в файл.
